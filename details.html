C/C++编程辅导

 

C++ Assignemnt

Objective:

Use a Linked List and a set of Derived Classes from a Base Class to demonstrate polymorphism in C++. To do so, alter linked-list source code to run a family of mobile phone derived classes, and create a menu to demonstrate the functionality.  Students are allowed  to work in groups or cohorts. For ease of submission, please e-mail the instructor the members of your cohort before the due date.

Description:

The Linked List

 

Sample code is available to create a linked list on Blackboard.  Alter this code to create a linked list capable of handling the derivation tree of a series of mobile phones. The base class is mobile phone.

 

The linked list must be an independent class, and must have the following data and functions:

  • MobilePhone * head – a pointer to the first location in the linked list.
  • MobilePhone * tail – a pointer to the last location in the linked list.
  • int size- an integer that has the current size of the linked list.
  • Add Phone-A member function that will add a Mobile Phone or one of its bases to the end of the list.
  • RingPhones-A member function that will traverse the list, and call the virtual function Ring() for each phone in the linked list.
  • LinkedList() Constructor- This public constructor will set the head and tail pointers to NULL
  • Clear -This public function will set the head to null, set the tail to null, and the size to 0.

 

 

 

Node Classes

The Base Class is Mobile Phone, and the family of classes has three derived classes.

Base Class: Mobile Phone

 

Member Data

The Moblie Phone has a protected data field called TelephoneNumber, which needs to be in the form of a string that contains 10 digits.

The Mobile Phone has a protected data field called RingTone that stores a string that represents the ring tone.

 

Member Functions

The Mobile Phone has a constructor, which will initialize the TelephoneNumber to “000-000-0000” and set the ringtone string to “RING!”

The Mobile Phone has a public function called “Show Number” to display its current telephone number.

 

Virtual Functions

The Base Class Mobile Phone has a virtual function “Ring()”, which will output the telephone number to the screen once, and then output the ringtone to the screen for a fixed number of times.  For the base class, the ringtone will ring 5 times.

 

Derived Classes: Goggle Phone, Doors Phone, Robot Phone

 

The three derived classes are Goggle Phone, Doors Phone and Robot Phone. Each in turn inherits from the Mobile Phone.  Each of the derived classes will each have its own Ring() function, which will include a number of repetitions. The number of repetitions is performed by a loop. When the phone rings, the output is printed to the screen

a.Prompt the user for the type of phone

4.Clear the linked list of all phones

5.Quit the system
